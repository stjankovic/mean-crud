## How to install

First clone the repository.
After that the next step is to install everything
`cd mean-crud`
`cd frontend`
`npm install`
`cd ..`
`cd server`
`npm install`

Simple as that

## How to run
To start the server
`cd server`
`npm run server`

To start the frontend
`cd frontend`
`npm serve`
